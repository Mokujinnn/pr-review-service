## **Вопросы/Проблемы при реализации сервиса:**


В документации, в эндпоинте `/team/add` есть противоречие между созданием команды и обновлением её пользователей:

- В summary сказано "создаёт/обновляет пользователей" - это относится к **пользователям**
- Код 400 "Команда уже существует" - относится к **команде**

В моей реализации, если команда уже существует, возвращается ошибка и пользователи не обновляются.
То есть, пользователи создаются только если команда не существует и после этого не могут быть обновлены.
